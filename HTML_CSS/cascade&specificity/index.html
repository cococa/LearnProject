浏览器将优先级分为两个部分：HTML 的行内样式和选择器样式
1. 行内样式没有选择器，因为他们直接作用于所在的元素，不建议在行内样式中用 !important， (行内的选择器优先级最高， 除非用!important) 2. 选择器优先级
优先级的计算是基于选择器的组成部分的权重。基本上，优先级可以通过以下几个维度来计算：
  • 1. 内联样式（Inline styles）：style="..."，优先级最高。
  • 2. ID 选择器（如 #id）：优先级高。
  • 3. 类选择器、伪类选择器（如 .class 和 :hover）：中等优先级。
  • 4.元素选择器、伪元素选择器（如 div 和 ::before）：优先级最低。
