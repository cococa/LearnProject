border 一般有下面三个属性：
border:width
border:style [ solid | dashed | dotted | double ]
border:color
当然也可以见写成 border : width style color;
其中最重要的属性就是 style ，如果不设置 style ，是没有办法显示border 的， 默认的width 为3-4px, 默认的color 是字体的颜色，默认的 style 则是 none


首先来看 css3 中的 border-color
1: border-color 可以拆分为 border-x-colors [x= top|right|bottom|left] (注意是带 s 的)
2: 在 css3 中 ，border-color 没有成为标准，所以要让浏览器正常工作，必须要带上前缀，可以用 webpack 自动生成


再来看 border-image
border-image : none | [number | percentage] {1,4} ?[ stretch | repeat |round ]
1: none 表示无边框背景
2: image 表示背景图片，就是图片的 url
3: number 边框或背景图片的大小，单位是px .可以设置 1-4个值
4: percentage 和number 一样。单位是百分比
5: 铺放方式，默认是 stretch，会拉伸背景 | 重复边框背景，repeat | 平铺边框背景 round
