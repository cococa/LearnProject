white-space 主要用来申明建立布局的过程中如何处理元素中的空白符 
		1. normal  空白处会被浏览器忽略
		2. pre 空白处会被扣留，类似与 pre 标签 ， 
		3. nowrap 文本不会换行，文本会在同一行上， 直到碰到换行标签
		4. pre-line 合并空白符序列，但保留换行符
		5. pre-wrap  保留空白符序列，但是正常进行换行
		6. inherit 继承父元素

在下面的文字中， this 前面有很大的一个空格

normal this is test
pre this is test
nowrap this is test
pre-line this is test
