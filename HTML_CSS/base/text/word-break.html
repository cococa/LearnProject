word-break 属性用来决定自动换行的处理方法，通过具体的属性设置，不仅可以实现半角空格或连字符后面的换行，还可以让浏览器实现任意位置的换行
看下基本的语法
normal : 默认值，中文到边界换，英文从整个单词换行
break-all : 可以强行截断英文单词，达到词内换行的结果
keep-all ：不允许字断开，

下面的例子中，左边为 break-all 右边为 keep-all ，可以看到 break-all 就是强行截断换的，而keep-all 在中文下 根据标点，英文则是证行换，会撑破容器
