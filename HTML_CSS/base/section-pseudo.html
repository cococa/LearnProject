2.1新增的动态伪类
1. :focus 当前拥有输入焦点的元素
2. :hover 鼠标指针停留的元素
3. :active 用户输入激活的元素
动态伪类可以用应用到任何元素
1: element:first-child 选择第一个子元素，如果还有子元素，则会被嵌套选择中 (IE6不支持)
2: element:first-letter 选择首字母 （css2中只能用块级元素和a标签） 具体选择的属性要去查表，伪元素选择器放在最后
3: element:first-line 选择第一行 （css2中只能用块级元素和a标签）
伪类可以与 CSS 类配合使用：
1. a.red:visited {color:#FF0000;}
