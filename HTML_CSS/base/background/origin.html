基本语法: background-origin:padding-box || border-box || content-box
这个属性可以根据自己的需求来改变背景图片的 background-position 起始位置
padding-box : (默认值） 从padding 的外边缘开始显示背景图片
border-box : 从 border 的外边缘开始显示背景图片
content-box : 从 content 的外边缘开始显示背景图片
查看例子：注意看下面三个块中白色图片的位置

background-origin: padding-box;
background-origin: border-box;
background-origin: content-box;
