属性选择器
从语法上来说，[]前可以带 element ，也可以不带 1: element[attr] 所有带 attr 属性的元素，都会被选择到 例：div[id]
2: element[attr="val"] 所有带 attr 属性，并且值是val的元素，例 div[class="large"] a[href="http://www.baidu.com"]
3: element[attr~="val"] 所有带 attr 属性，并且值中包含val的元素，一般class 用的比较多， 例：div[class~="red"]

css3 新增属性选择器
1: [att*=val] 选择器 att 属性中包含 val 的，都会被选择
2: [att^=val] 选择器 att 属性以 val 开头的，都会被选择
3: [att$=val] 选择器 att 属性以 val 结尾的，都会被选择
